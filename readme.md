# Sublime CSS Compiler

## Build from source

Linux:
```bash
chmod a+x ./build.sh
./build.sh
```

Windows:
```bash
./build.sh
```

## Using

``` bash
# for linux
bin/linux/sublimec style.css
# for windos
bin/win32/sublimec style.css
```
After, you can find near your style.css compiled css file
